# Data Analyst Assignment

## Objective
This project demonstrates data analysis skills using SQL, Excel, and Python.

## Tools Used

- SQL (MySQL)
- Microsoft Excel
- Python 3

---

## SQL Tasks

Hotel System:
- Last booked room per user
- Total billing for a given month
- Bills greater than 1000
- Most and least ordered items
- Second highest bill per month

Clinic System:
- Revenue by sales channel
- Top customers
- Monthly profit and status
- Most profitable clinic per city
- Second least profitable clinic per state

---

## Excel Tasks

- Used XLOOKUP and VLOOKUP for data mapping
- Compared date and time using INT() and HOUR()
- Created helper columns for analysis

---

## Python Tasks

Time Conversion:
Convert minutes into hours and minutes

Remove Duplicates:
Remove duplicate characters using loop
